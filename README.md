# Iter Catholicum ✝️

> **A bridge between tradition and technology.**

**Iter Catholicum** is a premium, high-performance liturgical companion designed for the modern Catholic life. It provides a majestic interface for navigating the General Roman Calendar and reading the Holy Scriptures in Latin and Spanish, combining rigorous technical precision with a dignified, clerical aesthetic.

## 🏛️ Project Philosophy

The project is built on the pillars of **liturgical accuracy**, **theological tradition**, and **technological excellence**. Inspired by the beauty of medieval manuscripts and the sobriety of ecclesiastical art, **Iter Catholicum** offers a professional-grade tool for the faithful to connect with the timeless rhythms of the Church.

---

## ✨ Key Features

### 📅 Advanced Liturgical Calendar
- **Powered by `romcal`**: Utilizing the latest development branch for maximum liturgical precision.
- **Bilingual Interface**: Seamlessly switch between Spanish and Latin (_lingua ecclesiastica_).
- **Monumental Design**: Premium typographic scale with monumental headers that harmonize the entire experience.
- **Roman Numerals**: Idiomatic Latin numbering for weeks and seasons (e.g., _Hebdomada II Quadragesimæ_).

### 📖 Scriptural & Lectionary Engine
- **Public Domain Only**: Exclusively hosts historical, copyright-free translations (Vulgata 1592 & Torres Amat 1823).
- **Advanced Reference Parser**: Robust logic that handles complex, non-contiguous lectionary citations (e.g., *Sal 65, 1-3a.4-5.6-7*).
- **Multi-level Navigation**: Professional breadcrumb system that organizes Testaments, Groups, and Books in an intuitive hierarchy.
- **Parallel Reading**: Instant access to parallel translations via interactive pop-ups.

### 🔍 Optimized Architecture
- **Intelligent Code-Splitting**: Optimized build process that isolates large biblical and lectionary datasets into separate, lazy-loaded chunks.
- **High Performance**: Designed to deliver a fast, responsive experience even when handling megabytes of scriptural data.
- **TTS Integration**: Built-in Text-to-Speech support for reading prayers and lectionary texts.

## 🚀 Getting Started

### Prerequisites
- Node.js (v20+)
- npm

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/danloi2/itercatholicum.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start development server:
   ```bash
   npm run dev
   ```
