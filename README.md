# Iter Catholicum ✝️

> **A bridge between tradition and technology.**

**Iter Catholicum** is a modern, high-performance liturgical companion designed for the Catholic life. It serves as a fluid interface for navigating the General Roman Calendar and reading the Holy Scriptures in Latin and Spanish, combining technical precision with a premium, clerical aesthetic.

## 🏛️ Project Philosophy

The project is built on the pillars of **liturgical accuracy**, **theological tradition**, and **technological excellence**. Inspired by the beauty of antique manuscripts and the sobriety of ecclesiastical art, **Iter Catholicum** offers a bridge for the modern faithful to connect with the timeless rhythms of the Church.

---

## ✨ Key Features

### 📅 Advanced Liturgical Calendar

- **Powered by `romcal`**: Utilizing the latest development branch for maximum precision.
- **Bilingual Interface**: Seamlessly switch between Spanish and Latin (_lingua ecclesiastica_).
- **Liturgical Accuracy**: Full support for feast ranks, proper colors, and cyclical seasons.
- **Roman Numerals**: Idiomatic Latin numbering for weeks and seasons (e.g., _Hebdomada II Quadragesimæ_).

### 📖 Scriptural Bridge

- **Public Domain Only**: Exclusively hosts historical, copyright-free translations.
- **Vulgata Clementina (1592)**: The definitive Latin text for centuries.
- **Biblia Torres Amat (1823)**: A classic, venerable Spanish translation.
- **Immersive Reading**: Typography features such as historical drop caps and ivory paper textures.

### 🔍 Search & Devotional Tools

- **Smart Search**: Instant, accent-insensitive search for biblical passages and liturgical events.
- **Prayers & Mass**: Integrated collections for daily prayer and liturgical follow-up.
- **Daily Context**: Instant summary of today's liturgical rank and obligation directly on the home page.

## � Getting Started

### Prerequisites

- Node.js (v18+)
- npm / yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/danloi2/itercatholicum.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start development server:
   ```bash
   npm run dev
   ```
